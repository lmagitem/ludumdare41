v1.4 (11 ноября 2014):
* добавлен новый параметр - Shuffle для получения "прыгающего" ритма;
* теперь любой параметр (BPM, Shuffle, количество треков и шагов) можно изменить в Главном меню -> Song Settings;
* iOS: поддержка Audiobus 2 и IAA (частично);
* исправлены ошибки.
v1.3 (18 февраля 2014):
* добавлена возможность ручного изменения ориентации экрана (только для Android): Preferences -> Interface -> Angle;
* в файловый браузер добавлены новые функции: Remove, Rename, Cut, Copy, Paste, Create Directory;
* файловый браузер теперь отображает размер файлов;
* новая композиция - Example10;
* исправлены ошибки.
v1.2.3 (21 мая 2013):
* поддерживаемые звуковые системы: Audiobus, ALSA, JACK (iOS and Linux), SDL, OSS, DirectSound, MME, ASIO;
* дополнительные настройки звука и графики в Главное меню -> Preferences -> Audio & Graphics;
* дополнительные фоновые картинки в Главное меню -> Preferences -> Background;
* теперь любой загруженный WAV файл будет играть с оригинальной скоростью на первой ноте 6ой октавы;
* улучшенный визуализатор;
* исправлены ошибки.

* первая версия.
